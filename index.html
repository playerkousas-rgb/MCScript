MC 提詞器 使用說明

使用步驟：

  1. 點擊「匯入講稿」，選擇你的 .txt 講稿
  2. 點擊右上角角色按鈕，輸入你的角色（例如：司儀）
  3. 點擊「開始語音跟隨」

如何上傳講稿（iPhone）：

  1. 先把 .txt 講稿傳到 iPhone（AirDrop / Telegram / WhatsApp）
  2. 打開「檔案」App，找到講稿
  3. 長按講稿 → 分享 → 複製到提詞器
  4. 或點擊「匯入講稿」後，選擇「瀏覽」→ 找到檔案

效果：

  • 青色框 = 你要讀的句子
  • 黃色背景 = 現在進行到的句子
  • 語音自動捲動，雙手不用動

注意：

  • 建議使用 Safari 開啟
  • 第一次會要求麥克風權限
  • 可隨時更改角色
