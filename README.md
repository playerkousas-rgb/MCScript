# MC 提詞器 Pro

網頁版司儀提詞器，支援語音自動跟隨。

## 部署到 Vercel（最簡單）

1. 把整個 `mc-teleprompter` 資料夾上傳到 GitHub（新建 repo）
2. 去 [vercel.com](https://vercel.com) 登入
3. 點擊 **New Project** → 選擇你的 GitHub repo
4. 點擊 Deploy

部署完成後，你會得到一個網址（例如：`mc-teleprompter.vercel.app`）

## 使用方法

1. 用 Safari 打開 Vercel 網址
2. 點擊右上角「匯入講稿」
3. 上傳你的 `.txt` 講稿
4. 點擊角色按鈕，輸入你的角色名稱（例如：司儀）
5. 點擊「開始語音跟隨」

## 加到手機桌面（像 App 一樣用）

1. 在 Safari 打開網頁
2. 點擊分享按鈕（方框+向上箭頭）
3. 選擇「加入主畫面」
4. 點擊「加入」

之後就可以直接從桌面打開，像 App 一樣使用。

## 功能

- 支援 `司儀：`、`司儀:` 格式
- 可輸入任何角色名稱
- 青色框 = 你要讀的句子
- 黃色 = 目前進行到的句子
- 語音自動跟隨
